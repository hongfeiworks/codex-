# instruction-reflector-zh

请回顾当前会话和最近暴露出的协作问题，检查并优化 Codex 的全局或项目级指令。

附加要求：

`$ARGUMENTS`

## 工作范围

优先检查这些内容：

- `AGENTS.md`
- 当前项目里是否还有更细的约束文件
- 最近几轮对话中反复出现的问题

## 重点关注

1. 回复风格是否稳定
2. 中文注释、中文文档要求是否被满足
3. 是否还在过度抽方法、过度并行或跑偏重构
4. Windows、UTF-8、BOM、换行相关规则是否够明确
5. 危险操作确认机制是否覆盖到位
6. 是否有规则写得太空、太重或彼此打架

## 工作方式

1. 先基于实际对话和实际问题找证据，不要空想。
2. 每条建议都要说明：
   - 当前问题是什么
   - 建议改哪一段
   - 改完会带来什么效果
3. 除非我明确要求你直接改，否则先给建议，再等我确认。
4. 建议尽量小而准，不要为了“完整”重写整份指令。

## 输出格式

1. 发现的问题
2. 对应建议
3. 建议修改的文本
4. 预期效果
